***FeraLab***
=============

Reborned Hybrid ROM and Kernel for SonyEricsson x10i/a
By FeraVolt.

== Installation ==
1) Get and install FeraKernel v20+ (no need for drivers package - it's built-in)
2) Copy your ROM .zip to SDcard
3) Boot in CWM6 recovery and install ROM .zip following installation wizard
4) Reboot & wait not less than 2 minutes after complete boot-up
5) Reboot again and be happy!
