***FeraLab***
=============

Reborned Hybrid ROM and Kernel for SonyEricsson x10i/a
By FeraVolt.

== Installation ==
(If you have locked bootoader - ignore first step)
>> Extract 'FeraKernel.ftf' from firmware package to '/flashtool/firmwares/' folder & Flash kernel from FlashTool. 
>> Boot-up to CWM6 & just straightly install ROM following AROMA installation wizard. No need to manually mout or wipe something - everything is automized.
>> After installation, boot your phone, pass setup wizard and wait 2 minutes. Reboot again.
>> On second boot wait not less than 2 minutes again & reboot.
>> Done. Do a reboot after you configured your phone.
